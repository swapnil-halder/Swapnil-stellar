# Stellar Booking UI

A React + Vite application for booking and reservation flows on the Stellar blockchain.

This repo contains a modern landing page and a dashboard for:

- connecting a Stellar wallet via Freighter
- creating, booking, cancelling, and completing time slots
- querying slot information and viewing action output

## Development

Run locally with:

```bash
npm install
npm run dev
```

Then open the URL shown by Vite.
